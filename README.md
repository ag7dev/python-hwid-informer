Its a simple desktop application built with `customtkinter` that provides detailed system information such as CPU, RAM, disk usage, OS version, and more. Additionally, it allows users to retrieve their HWID (Hardware ID) and copy system information to the clipboard with just one click.

## Features
- Displays system information such as:
  - CPU, Cores, Threads
  - RAM usage
  - OS version, hostname, IP address
  - Disk usage (total and used)
- Displays the unique Hardware ID (HWID) of the system.
- Allows users to copy system information (CPU, RAM, HWID, etc.) to the clipboard with a simple button click.
- Supports theme selection (Light, Dark, or System default).
